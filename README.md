# Ronak VJ Linear Website

Interactive React portfolio website for Ronak VJ. The site presents the career tree as a seven-element linear journey with expandable roles, project placeholders, dark/light theme support, and responsive sections for About, What I Do, My Journey, and Contact.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Preview production build

```bash
npm run preview
```

## Main files

- `src/App.jsx` - main website UI and interactions
- `src/data/careerTree.js` - stages, roles, and project placeholders
- `src/index.css` - Tailwind import and theme styling
- `vite.config.js` - Vite configuration
